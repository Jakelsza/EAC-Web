How to use the companion

          
Everything you need for a clean Abyss run, in order.


          

            
1 · One-time setup

            
On first launch the wizard asks for your EVE Gamelogs folder — usually
              Documents\EVE\logs\Gamelogs. This is how the companion sees NPCs:
              it only ever reads the log files EVE writes to disk. No memory reading, no game hooks,
              nothing that touches the client. You can change the folder any time under Systems.

            
Folder access and the floating overlay work best in Chromium (Chrome / Edge). Firefox is supported for all other features;
              NPC detection requires you to load the Gamelogs folder via the folder picker at the start of each room (Firefox cannot watch files for live changes).
              You can change the folder any time under Systems.

          


          

            
2 · The run timer

            
Pick your Tier and Weather, type your cargo's estimated value into Before,
              and hit ▶ Start as you take the gate. The ring counts down the 20-minute Abyss limit and
              turns red under five minutes. Audio cues fire at 5:00, 1:00 and 0:00 (toggle in Systems).

            
Tap R1 ✓ / R2 ✓ / R3 ✓ as you clear each room. Each split shows the time spent
              inside that room, so you can see exactly which room costs you. Marking R3 stops the
              clock and waits for your loot total.

          


          

            
3 · NPC detection

            
During a run the companion watches the log and lists every hostile it sees, colour-coded by
              the trouble they bring:

            
    
              
  • Scram — warp scrambler, you're held
    • 
              
  • Neut — energy neutraliser, drains your cap
    • 
              
  • Web — stasis web, slows you
    • 
              
  • Damp/Dis — sensor damp / tracking disruption
    • 
              
  • Logi — remote reps; kill these first
    • 
              
  • Paint — target painter
    • 
              
  • Gold border — room boss · the gold chip is the latest spawn
    • 
            

            
The same list shows in the workflow panel and the overlay. It resets each room. If your fit
              has no drones, switch off I use drones in Systems to hide the drone-repair flag.

          


          

            
4 · The overlay

            
Overlay ▶ opens a floating always-on-top window (Picture-in-Picture) you can drop over
              EVE in windowed-fullscreen. It carries the timer, detected NPCs and the run controls, and
              resizes freely — everything scales to fit. Clicking it briefly takes focus from EVE; one click
              back into the game returns it.

          


          

            
5 · Loot, history & stats

            
After the run, enter your cargo's new value in After and hit ✓ Submit & End
              (or ■ End any time to log a partial run). The difference is your loot for that run.

            
History lists every run; rows that hold an all-time record for their tier + weather are
              highlighted. Stats breaks down ISK/hour and shows, from your own runs, which tier +
              weather pays best — so you know what to grind.

          


          

            
6 · Sessions

            
Planning a block of runs? In Stats, name a session and set how many runs you intend to
              do. The companion tracks just those runs, shows your progress, and gives you a clean summary
              when you finish. Past sessions are saved so you can compare a Tuesday grind to a Wednesday one.

          


          

            
7 · Your own NPC data

            
The bundled dataset is a small starter set. Import your full
              npc_dataset.json under Systems → Import dataset — same schema,
              snake_case kill_priority included. The NPC Library is fully searchable
              and filterable by faction.

          

        

      


      

        

          
EVE Abyss Companion

          
v2.1.0 · Web Edition — “Built by capsuleers, for capsuleers.”

          
A free companion for Abyssal Deadspace runs. It reads your EVE Online game log files from disk to
            detect NPCs — no memory reading, no game hooks, no injection, no EVE API calls. Everything runs and
            stays on your machine: settings, history and your NPC dataset never leave this file.

          
Requirements: a modern browser (Chrome, Edge, or Firefox) opened directly on this file.
            The floating always-on-top overlay and seamless live log monitoring use APIs only available in Chromium (Chrome / Edge).
            Firefox is fully supported for all other features; NPC detection works via the folder picker with a one-time file load per room.

EVE Abyss Companion keeps you alive inside the Abyss: it runs your 20-minute death timer,
        spots hostile NPCs from your combat log, and tracks what each run actually paid.

      
    
        
  • 20-minute filament timer with room splits
    • 
        
  • Automatic NPC detection with EWAR-coded tags
    • 
        
  • Loot delta tracking, run history and ISK/hour stats
    • 
        
  • Always-on-top overlay for use over EVE
    •
